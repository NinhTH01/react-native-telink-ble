# react-native-telink-ble

Telink BLE bridge for React Native

## Installation

```sh
npm install react-native-telink-ble
```

## Usage

```js
import TelinkBle from "react-native-telink-ble";

// ...

const result = await TelinkBle.multiply(3, 7);
```
